Structured Query Language (SQL) Command:

  • SELECT -- Select which colum you wish to display
  • DISTINT -- Do not display any duplications
  • AS -- Display custon name to colum
  • FROM -- From which table you wish to display SELECTED colum
  • SELECT * FROM [Table Name] -- Display all colum from SELECTED table
  • SELECT TOP(3) * FROM [Table Name] -- Display top 3 colum from SELECTED table
  • SELECT * FROM [Table Name] WHERE [Colum Name] = '[Selected Data]' -- Display only [Selected Data] from specific [Colum Name]
