Data types in SQL:

  • INT -- whole numbers
  • DECIMAL(M,N) -- Deciaml numbers - Exact Value
  • VARCHAR(l) -- String of text of lenght l
  • BLOB -- Binary large object, store large data
  • DATE -- YYYY-MM-DD
  • TIMESTAMP -- YYYY-MM-DD HH:MM:SS - used for recording

Structured Query Language (SQL) Command:

  • SELECT -- Select which colum you wish to display
  • DISTINT -- Do not display any duplications
  • AS -- Display custon name to colum
  • FROM -- From which table you wish to display SELECTED colum
  • SELECT * FROM [Table Name] -- Display all colum from SELECTED table
  • SELECT TOP(3) * FROM [Table Name] -- Display top 3 colum from SELECTED table
  • SELECT * FROM [Table Name] WHERE [Colum Name] = '[Selected Data]' -- Display only [Selected Data] from specific [Colum Name]
  • SELECT * FROM KCC.DBO.Customers WHERE CustomerName <> 'Tres Delicious' -- Display ever [Selected Data] exept specific [Colum Name]
  • -- This is comment in SQL /*This is comment in SQL*/
  • SELECT * FROM KCC.DBO.Customers WHERE State NOT IN('WI', 'NY', 'UT'); -- Display everything exept State WI, NY, UT -- or if you leave just IN will be other way round
  • SELECT * FROM KCC.DBO.Customers WHERE CustomerName LIKE 'A%'; -- Display me all Customer colum with beggining letter of A
  • SELECT OrderID,
    OrderDate,
    OrderTotal,
    CustomerName,
    Phone
    FROM KCC.dbo.Orders
    JOIN KCC.dbo.Customers on KCC.dbo.Orders.CustomerID = KCC.dbo.Customers.CustomerID
    -- This is how to join two table and display colums from two diffrent tables
  • ORDER BY [Colum Name] -- display number by order from specific colum
  • ORDER BY OrderTotal DESC -- Same as above just in descending order
  • SELECT * FROM KCC.dbo.Orders
